# Process-Scheduler
This project is a CPU scheduler which used the following scheduling algorithms using C.
========================================================================================
1. Preemptive Highest Priority First (PHPF).
2. Shortest Remaining time Next (SRTN).
3. Short Job First (SJF).
4. Round Robin (RR).
